The Element Antimony!

The element Antimony is a lustrous gray metalloid with the atomic number 51 and the symbol Sb.
Antimony is found in nature mainly as the sulfide mineral stibnite and has been used in medicines and cosmetics
all the way back to egypt as "Kohl".
